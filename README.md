# Autotools

Demo some autotools usage with modern C++ code.

A lot of what's done here is based off of the [Bitcoin Core](https://github.com/bitcoin/bitcoin) build system.

## Usage

```bash
./autogen.sh
./configure
make

./src/basic_binary
Basic Autotools 0.0.1 Binary
API Version: 1
Calling some_func(2): 40
```
